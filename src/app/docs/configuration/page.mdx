# Configuration

Diwan can be configured using a `toml` file.

## Config Path

The default configuration file is located at `~/.config/diwan/config.toml`.

## Example Config

```toml filename="~/.config/diwan/config.toml"
[editor]
theme = "tokyonight"
line_numbers = true
tab_size = 4

[lsp]
enable = true
```
