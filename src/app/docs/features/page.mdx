# Features

Diwan comes with many features out of the box.

- **Modal Editing**: Vim-like modal editing for maximum efficiency.
- **LSP Support**: Language Server Protocol support for autocompletion and diagnostics.
- **Tree-sitter**: Syntax highlighting powered by tree-sitter.
- **Lightning Fast**: Built with Rust for optimal performance.
- **Customizable**: Fully customizable via TOML.
