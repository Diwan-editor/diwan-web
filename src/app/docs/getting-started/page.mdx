# Getting Started

Diwan is a modern, fast, and extensible terminal editor built with Rust.

## Installation

To install Diwan, run:

```bash filename="install.sh"
curl -proto '=https' --tlsv1.2 -sSf https://diwan-editor.sh | sh
```

## First Steps

Once installed, you can start Diwan by typing `diwan` in your terminal.

```bash
diwan
```
